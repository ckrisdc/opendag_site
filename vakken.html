Welkom op de Curio info site voor de opendag

Hier kan je vinden welke vakken je zult hebben en wat het een beetje inhoud

Je hebt in het eerste jaar twee blokken met voornamelijk dezelfde vakken maar, alleen in blok A heb je een extra vak dat als keuzedeel telt. Dit vak is digitale vaardigheden.

Je hebt in het eerste jaar de volgende vakken:

  • Nederlands
  • Engels
  • rekenen
  • web
  • pro
  • pra
  • native
  • digitale vaardigheden (alleen in blok A)

Nederlands, Engels en rekenen spreken voor zich en zijn de generieke vakken.

Bij web gaat het in de eerste plaats over web developement en je leert dus hoe je web applicaties maakt.

Pro is een vak waarin je leert hoe programmerenin z'n werk gaat. Bij pra ga je toepassen wat je leert bij pro, want pra staat voor praktijk.

Native is het vak waarin je leert hoe je apps maakt op basis van c# en python.

Digitale vaardigheden is een les in de week waarin je leert hoe je met verschillende programma's en situatioes om moet gaan. Het gaat ook over hoe je veilig met het internet om moet gaan.
